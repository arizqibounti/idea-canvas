# Neuroscience → Structure Mapping

> Researched March 2026. Which computational structures most closely resemble how the brain solves problems.

---

## Ranking: Most Brain-Like → Least Brain-Like

### 1. Causal Loop Diagrams (The Brain's Computational Architecture)

The brain IS feedback loops — not metaphorically, literally:

- **Predictive coding** (Karl Friston): Constant prediction→error→update loops between cortical layers. Top-down prediction meets bottom-up sensory data, error feeds back, prediction adjusts.
- **Excitation/inhibition balance**: Every neural circuit has excitatory (+) and inhibitory (−) polarity — exactly causal loop edge polarity.
- **~80% of cortical synapses are recurrent (feedback)**, not feedforward. The brain is overwhelmingly cyclic.
- **Attractor dynamics**: Problem-solving = brain settling into attractor states through loop traversal.
- **"Aha moments"**: Insight occurs when a dormant loop suddenly completes — gamma burst synchronization forming a new reinforcing loop.

### 2. Zettelkasten (The Brain's Memory System)

- **Hippocampal indexing theory** (Teyler & DiScenna): Hippocampus stores indices/links to distributed cortical representations — permanent IDs linking across contexts.
- **Sleep consolidation**: Hippocampus replays experiences, transfers links to cortex. Emergent clusters form schemas over time.
- **Spreading activation** (Collins & Loftus): Retrieving one concept activates related concepts across the entire network — bidirectional, cross-contextual.
- **Schema theory**: Prior knowledge structures = Zettelkasten clusters.

### 3. Graph of Thoughts (The Brain's Problem-Solving Procedure)

- **Default Mode Network (DMN)** generates multiple parallel candidates (divergent thinking = "generate" phase)
- **Convergence zones** (Damasio) aggregate from different perspectives (= "aggregate" phase)
- **Executive Control Network (ECN)** evaluates, refines, selects (= "refine + select" phase)
- **Neural competition**: Multiple hypotheses maintained simultaneously, compete via drift-diffusion until one wins
- **DMN↔ECN oscillation** during creative tasks (Beaty et al., 2018) = GoT's generate→evaluate loop

### 4. DAG

Captures hierarchical processing and multi-parent convergence — but acyclicity is fundamentally wrong for the brain.

### 5. IBIS

Question-driven exploration maps to curiosity circuits, but too tree-constrained for the brain's parallel search.

### 6. Toulmin

Maps to a single predictive inference but too linear for massively parallel, cyclic processing.

---

## Key Insight

The brain combines all three top structures:
- **Causal Loops** = how it processes (computational architecture)
- **Zettelkasten** = how it stores and links (memory system)
- **Graph of Thoughts** = how it creates (problem-solving procedure)
